# プレゼン資料作成ルール（熟練デザイナー＆プレゼンター準拠）

## 🎨 デザイン4原則（必須遵守）

### 1. 近接（Proximity）
- 関連する情報はグループ化して近くに配置
- 関連性のない情報は明確に離す
- グループ間には十分な余白を確保

### 2. 整列（Alignment）
- すべての要素に視覚的な結びつきを持たせる
- 基本は「左揃え」（可読性が最も高い）
- タイトルや図解は「中央揃え」も可
- 要素の開始位置を統一する

### 3. 反復（Repetition）
- 色、フォント、レイアウトパターンを全スライドで統一
- 見出しスタイル、箇条書きスタイルを一貫させる
- ルールを決めたら例外を作らない

### 4. コントラスト（Contrast）
- 重要な情報は視覚的に際立たせる
- フォントサイズ：タイトル32pt以上、本文18-24pt
- 色のコントラスト比を確保（背景と文字）

---

## 🎯 Presentation Zen 3原則

### 抑制（Restraint）
- 1スライド = 1メッセージ
- 箇条書きは最大3-5項目
- 装飾は最小限に

### シンプル（Simplicity）
- 余計なグラデーション、影、3D効果は使わない
- 色は3色まで（メイン、サブ、アクセント）
- 余白は「情報」として活用する

### 自然さ（Naturalness）
- 聴衆の視線の流れに沿ったレイアウト（Z型・F型）
- 話の流れと視覚的な流れを一致させる

## 📐 レイアウト仕様

### カラースキーム（例）
- プライマリ: #1A73E8（青系）or 会社カラー
- セカンダリ: #333333（ダークグレー）
- アクセント: 補色または明度違い
- 背景: #FFFFFF（白）or #F5F5F5（ライトグレー）

### フォント
- 日本語: Noto Sans JP
- 英語: Arial
- タイトル: 28-36pt、太字
- 本文: 18-24pt、標準
- 注釈: 12-14pt

### マージン・余白
- スライド端から最低20pxのマージン
- 要素間の余白は統一（8px, 16px, 24pxなど4の倍数）
- タイトルと本文の間は24px以上

## 🖥️ ビジュアルスタイル（sample_images準拠）

サンプル画像から抽出したモダンなダークテーマスタイル。

### 全体の雰囲気
- **ダークテーマ**: 全スライド暗い背景で統一
- **テック感のあるミニマルデザイン**: 余計な装飾なし
- **大胆なタイポグラフィ**: 数字やキーワードを大きく表示
- **十分な余白**: 情報密度は低く、呼吸するレイアウト

### カラースキーム
```
背景:
  - メイン背景: #0d1117（深いダークグレー）
  - 中央グロー: 放射状のサブトルな緑のグラデーション（rgba(78,205,196,0.1)程度）

テキスト:
  - プライマリ: #ffffff（白）
  - セカンダリ: #8b949e（ミュートグレー）
  - ラベル/カテゴリ: #4ECDC4（ミント/ティール）レタースペーシング広め

アクセント:
  - ハイライト: #4ECDC4（ミント/ティール緑）
  - 警告/損失: #DC143C（クリムゾンレッド）
  - 注意: #FFD700（ゴールド）

ライン/ボーダー:
  - 薄いボーダー: rgba(255,255,255,0.1)
  - アクセントライン: #4ECDC4
```

### タイポグラフィ
```
ラベル（カテゴリ表示）:
  - サイズ: 10-12pt
  - スタイル: 大文字、レタースペーシング 0.2em
  - 色: #4ECDC4（アクセント色）

ヒーロー数字/キーワード:
  - サイズ: 72-96pt
  - ウェイト: Bold
  - 色: #4ECDC4（アクセント色）

メインタイトル:
  - サイズ: 48-64pt
  - ウェイト: Bold
  - 色: #ffffff

サブタイトル/本文:
  - サイズ: 16-20pt
  - ウェイト: Regular
  - 色: #8b949e（ミュートグレー）

フッター:
  - サイズ: 12pt
  - 色: #8b949e
```

### レイアウトパターン

#### Hero/Impact スライド（数字を強調）
```
+------------------------------------------+
|  LABEL（小さく、アクセント色）              |
|                                          |
|  90%（巨大な数字、アクセント色）            |
|  キーワード（白、太字）                    |
|                                          |
|  説明文（グレー、小さめ）                  |
|                                          |
+------------------------------------------+
```

#### Closing/Q&A スライド
```
+------------------------------------------+
|                                          |
|           THANK YOU（ラベル）             |
|           Q&A（大きく白）                 |
|           サブテキスト（グレー）            |
|                                          |
|        [ボーダー付きボックス]              |
|         リソースリンク                    |
|                                          |
|  Presented by          日付              |
|  名前                   イベント名         |
+------------------------------------------+
```

### 装飾要素
- **グロー効果**: 背景中央に微かな放射状グラデーション
- **薄いボーダー**: ボックスは塗りつぶしではなく細い枠線
- **縦線ディバイダー**: Before/After比較などで使用
- **ボックスシャドウ**: 使用しない（フラットデザイン）

### CSSテンプレート例
```css
body {
  background: #0d1117;
  background-image: radial-gradient(ellipse at center, rgba(78,205,196,0.08) 0%, transparent 70%);

.label {

  color: #4ECDC4;
  text-transform: uppercase;
  letter-spacing: 0.2em;
}

  font-weight: bold;
  color: #4ECDC4;
}

.title {
  font-size: 48pt;
  font-weight: bold;
  color: #ffffff;
}

  color: #8b949e;
}

.box {
  border: 1px solid rgba(255,255,255,0.15);
  border-radius: 8pt;
  padding: 16pt 24pt;
  background: rgba(255,255,255,0.03);
}
```

### やってはいけないこと
- ❌ 明るい背景（白、ライトグレー）
- ❌ 派手なグラデーション
- ❌ ドロップシャドウの多用
- ❌ 情報の詰め込みすぎ
- ❌ 複数のアクセント色の同時使用
- ❌ 小さすぎるテキスト
- ❌ 緑を「競馬の芝」のような明るい緑にする（ティール/ミント系を使う）
